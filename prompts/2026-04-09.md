## 1

### 질문
`@src/store/toastStore.ts 가 왜 type error가 나는지 알려줘.`

### 답변
`src/store/toastStore.ts`의 주요 원인은 `zustand` import 방식 불일치입니다.

- 현재 코드: `import create from "zustand";`
- 에러 이유: 현재 설치된 `zustand` 타입 정의에서는 default export 호출 시그니처가 없어 `create(...)`를 함수로 인식하지 못함
- 1차 에러가 발생하면서 `set`, `state`, `message`, `id`, `toast` 등 매개변수 타입 추론이 연쇄적으로 깨져 `implicit any` 에러가 함께 발생
- 해결 방향: `import { create } from "zustand";`로 변경하면 타입 추론이 정상 복구됨
